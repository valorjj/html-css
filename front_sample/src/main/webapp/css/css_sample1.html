CSS Selector : where to apply

  1. *{ attr = "value" } : 모든 곳에 효과
  2. tag name { attr = "value" } : 모든 태그에 전역 효과
  3. .classname { attr = "value" } : 특정 클래스명 (자주 사용하는)에 효과
  4. #idname { attr= "value" } : 특정 id명 (특정 클래스, 특정 태그)에 효과
  5. 1. 동일한 효과가 있을 경우 우선순위 [ id > .class > tag(mark up) ]
  6. 2. class : 자주 사용되는 스타일에 적용한다.
  7. 3. id : 한 페이지에 한번만 사용한다.
